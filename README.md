# MindStack Extension 👻

MindStack (The Learning Ghost) is a browser extension that passively captures what you read and watch into your second brain. It tracks your reading context on the web and video segments on YouTube, sending them to your MindStack vault.

## Features

- **Passive Web Text Capture**: Automatically captures significant text sections you dwell on while reading articles.
- **YouTube Video Tracking**: Tracks YouTube video segments you watch and captures keyframes.
- **Manual Notes**: Jot down quick thoughts and save them directly to your active workspace.
- **Resource Uploads**: Upload files and images to your vault.
- **Live Feed**: See your real-time captures in the extension popup.
- **Zero-Friction Authentication**: Powered by Supabase.

## Setup & Installation

To run this extension locally or build it for your own use:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/mindstack-extension.git
   cd mindstack-extension
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key
   ```

4. **Build the extension:**
   ```bash
   npm run build
   ```
   *(Note: The build process uses `@crxjs/vite-plugin`. It may output a CJS deprecation warning, which you can safely ignore as long as the `dist/` directory is created).*

5. **Load the extension in Chrome:**
   - Open Chrome and navigate to `chrome://extensions`
   - Enable **Developer mode** in the top right corner.
   - Click **Load unpacked** and select the `/dist` directory generated by the build step.
   - Pin the 👻 icon to your toolbar!

## Tech Stack

- **Framework**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Extension API**: Chrome Manifest V3, `@crxjs/vite-plugin`
- **Auth**: Supabase

## Usage

1. **Log in** via the extension popup.
2. **Create or select a Workspace**.
3. **Start a Capture Session**.
4. Read articles or watch YouTube videos — MindStack will stealthily capture context in the background. A small green toast notification will appear bottom-right when a capture is successful.
5. End your session when you're done learning!
